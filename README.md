# Customer Churn Analytics Dashboard

## Project Overview
This project analyzes customer churn and revenue patterns using Power BI.  
The goal is to identify key drivers of customer churn and provide business insights through an interactive dashboard.

## Tools & Technologies
- Power BI
- SQL-style data analysis
- Data modeling
- Data visualization

## Dataset
The dataset contains customer information including:
- customer demographics
- subscription details
- tenure
- churn status
- revenue

The dataset includes thousands of records used to analyze customer behavior and churn trends.

## Dashboard Features

### KPI Overview
- Total Customers
- Churn Rate
- Revenue Metrics

### Customer Segmentation
- Churn by customer demographics
- Churn by tenure
- Customer distribution by region

### Trend Analysis
- Monthly churn trends
- Customer growth patterns

### Business Insights
The dashboard helps identify high-risk customer segments and patterns contributing to churn.

## Key Insights
- Customers with shorter tenure show a higher probability of churn
- Certain customer segments contribute disproportionately to revenue loss
- Churn trends vary across different customer groups

## Project Goal
The objective of this project is to demonstrate data analysis and visualization skills by transforming raw data into meaningful business insights.
